IFome

Esse é o projeto da minha equipe na disciplina de Engenharia de Software. Trata-se de um e-commerce desenvolvido para facilitar as vendas dos alunos no nosso Instituto Federal. Nossa equipe está focada em criar uma plataforma intuitiva e eficiente, onde os alunos poderão comprar e vender produtos de forma fácil e segura. O objetivo é proporcionar uma experiência de compra online conveniente, promovendo a interação entre os membros da comunidade acadêmica e incentivando o empreendedorismo entre os estudantes. Para alcançar esse objetivo, estamos utilizando tecnologias modernas de desenvolvimento de software, como JavaScript, Python, HTML e CSS, além de frameworks relevantes para garantir uma navegação responsiva e uma interface amigável. A segurança é uma prioridade em nosso projeto, e estamos implementando medidas robustas para proteger os dados dos usuários e suas transações.
